# PyCraft
파이썬과 Pyglet을 이용한 간단한 마인크래프트 입니다.

## **Original Source** : https://www.youtube.com/watch?v=NyBkFkAWnEo&t=2s  
## **fogleman/Minecraft** : https://github.com/fogleman/Minecraft

## 실행하는 법

```shell
pip install pyglet 
git clone https://github.com/Egnima/Pycraft.git
cd PyCraft
python Main.py
```
